About this site

Front End

The front end of this site is very basic Angular. I used ng2-pdfjs-viewer to create the pdf viewer for my cv and connected to a nest.js backend for the Nasa photos.

Front End Code

Back End

I created the back end using nest.js, a node.js framework. I used it to connect to a Nasa API that provides 4 pictures from their archives with explanations.

Back End Code

Deployment

I used Digital Ocean to create a pipeline to deploy both the front and back ends from the repository. Deployments are automatic with every change that I merge to master.
